# Recite ✨

A beautiful memorization app. Memorize anything — prayers, speeches, songs, poems, scripts — using 5 scientifically-proven practice modes and spaced repetition.

## Features
- 5 Practice Modes: Progressive Hiding, Fill in the Blank, First Letter, Meaning Recall, Full Recitation
- SM-2 Spaced Repetition tracking each line independently
- Progress tracking with mastery rings and streak counter
- Built-in user guide
- Dark, elegant design with gold accents
- Mobile-first, works on any device
- All data stored locally in your browser
